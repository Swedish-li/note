`~` 在 javascript 中是一个位操作符(bitwise NOT), 两个波浪号代表队一个操作数连续执行两次 “非” 的位操作。

由于 javascript 中的位操作会隐式的剑操作数转换为 32 位有符号的整数，所以这个操作符可以删除操作数十进制小数点之后的所有数字。

操作数可以是浮点数或者字符串，得到的结果是一个数字。这个操作等价于下面的代码:

```javascript
function(x) {
    if(x < 0) return Math.ceil(x);
    else return Math.floor(x);
}
```

x 的范围需要在 -(2^31) 和 2^31 - 1 之间，否则会产生溢出。
